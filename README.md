# ClawFi Documentation

Documentation site for the ClawFi crypto intelligence platform.

Built with [Astro](https://astro.build) and [Starlight](https://starlight.astro.build).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
